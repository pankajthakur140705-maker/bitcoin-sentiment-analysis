# Bitcoin Sentiment Analysis

## Objective
Analyze the relationship between Bitcoin market sentiment and trader performance.

## Project Structure
- notebook/analysis.ipynb
- report/report.md
- images/

## Key Findings
- Highest profitability was observed during Extreme Fear and Fear periods.
- Market sentiment has a measurable impact on trader performance.
- Sentiment can be used as an additional signal for trading decisions.
